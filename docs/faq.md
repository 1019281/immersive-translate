# 常见问题

## 如何更新最新规则

扩展本身会在使用的时候定时同步官方最新对网站适配的规则，你也可以手动立即同步最新的规则，浏览器点击扩展图标，打开弹窗后会自动检测最新适配规则并同步，油猴脚本同理。
